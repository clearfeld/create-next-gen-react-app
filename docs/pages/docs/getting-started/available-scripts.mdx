# Available Scripts

import { Tab, Tabs } from 'nextra-theme-docs'

In the project directory, you can run the following script commands:

## Turbo

All script commands listed below on this page can be ran with turbo, by prefixing the command with `t:`.

For example if you want to run the `test` script with turbo you can use the following command format.

<Tabs items={['npm', 'pnpm']}>
  <Tab>
    ```bash copy
    npm run t:test
    ````
  </Tab>
  <Tab>
    ````bash copy
    pnpm t:test
    ````
  </Tab>
</Tabs>

## Scripts

### `start`

<Tabs items={['npm', 'pnpm']}>
  <Tab>
    ```bash copy
    npm run start
    ````
  </Tab>
  <Tab>
    ````bash copy
    pnpm start
    ````
  </Tab>
</Tabs>

Runs the app in the development mode. Your browser will automatically open to http://localhost:3000.

The page will reload if you make edits.
